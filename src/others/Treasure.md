# 宝藏工具
[shell](https://nilesoft.org/)：一个小软件，修改windows右键，很实用包括转至一些系统文件夹，按shift以管理员运行终端等

[snipaste](https://zh.snipaste.com/)：F1截图 F3贴图，可以让截图悬浮在任意位置，方便做些对比

[Wechat2RSS](https://wechat2rss.xlab.app/list/)：公开提供部分公众号RSS，支持私有部署
